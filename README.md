# NewsApp
An Android app built with Jetpack Compose and the latest android libraries.
## Installation
Steps to set up the project locally:
1. Clone the repository.
2. Open in Android Studio.

## Technologies Used
- Kotlin
- Firebase
- JetPack COmpose
